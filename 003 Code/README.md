## ⚠️ 주의사항

본 프로젝트는 산학연계 캡스톤 디자인 프로젝트와 관련하여 다음의 주의사항을 준수합니다.

1. **Open Source SW 저작권 명시:** 본 프로젝트에 사용된 오픈소스 소프트웨어의 저작권 및 라이선스는 하단 `Open Source Licenses` 항목에 명시되어 있습니다.
2. **기업 기밀 데이터 보호:** 기업의 기밀이 담긴 데이터(API Key, 내부 문서, 고객 정보 등)는 소스 코드에 포함되지 않았습니다.
3. **책임 고지:** **기업 기밀 데이터가 Github에 공개되었을 시의 책임은 공개한 학생에게 있음을 인지하고 있습니다.**


## 🛠 설치 및 실행 (Installation & Usage)

#### Frontend 의존성 설치
```bash
npm install
````

### 프로젝트 실행

#### Frontend
```bash
# 기본 실행
npm start

# Android
npm run android

# iOS
npm run ios

# Web
npm run web
```
